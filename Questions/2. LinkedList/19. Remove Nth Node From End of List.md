## 思路：
1. 传统解法：找到倒数n个节点，删除它，
   1. 遍历两遍： 1得到size，2 size - n
2. 双指针遍历一遍：快慢指针
```java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        // 本质是找到倒数n个节点，删除它，
        // 在链表中如何找到倒数n个节点？仅用 O(n)复杂度一次遍历？
        // 快慢指针
        // 还是利用了线性遍历访问的，链表的连续性的基础之上。
        ListNode dummy = new ListNode(-1, head);
        ListNode slow = dummy; // 待删除节点之前的一个节点
        ListNode fast = dummy;

        while (n != 0) { // fast 节点先移动n步
            fast = fast.next;
            n--;
        }

        // 找到倒数第n个节点
        while (fast.next != null) { // fast 下一个节点为空时，slow的下一个节点正好是待删除节点
            slow = slow.next;
            fast = fast.next;
        }

        // 删除操作
        slow.next = slow.next.next;
        return dummy.next;
    }
}
```
