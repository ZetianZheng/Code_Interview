## 审题（关键词） 
二分法+找插入index

## 初始思路  
二分法确认到最终位置，如果不为target值，则返回该index

## 考点  
二分法的循环不变量，
如果是左闭右闭的话，则index最终为mid位置

## 解法  
```java
class Solution {
    public int searchInsert(int[] nums, int target) {
        int l = 0, r = nums.length - 1;

        // 左闭右闭，如果没找到，则在mid位置插入
        while (l <= r) {
            int mid = (l + r) >> 1;
            if (nums[mid] < target) {
                l = mid + 1;
            } else if (nums[mid] > target) {
                r = mid - 1;
            } else {
                return mid;
            }
        }

        return l;
    }
}
```
